## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-bootstrap-stt`
- `allow-download-model`
- `allow-transcribe-file`
- `allow-stt-health`
- `allow-setup-record-transcribe-pipeline`
- `allow-initialize-recorder-runtime`
- `allow-start-recording`
- `allow-stop-recording`
- `allow-set-hotkey-bindings`
- `allow-capture-hotkey`
- `allow-get-runtime-state`
- `allow-set-output-destination`
- `allow-get-output-destination`
- `allow-set-overlay-mode`
- `allow-get-overlay-mode`
